Algorithm instruction

Algorithm - user defined python source file which contains functions initialize, handle_data and optionally functions load, save and analyze
Initialize - function for learning
Handle_data - function for buying and selling and optionally for online learning
Load - function which defines the way algorithm training parameters are loaded. If not specified, default function just pickles out result variable from train parameters file if it's uploaded
Save - function which defines the way algorithm training parameters are saved. If not specified, default function pickles result to train parameters file if it's uploaded:
Analyze - function for depicting the results of testing algorithm. It should plot everything needed and return figure object. Default function would plot graph of portfolio value (run Quick Start for an example)
