Dataset format instruction

Notes on dataset format:

Price - there can be column price in provided dataset. When calling function order() zipline "buys" stocks according to the column. If not provided column close will be used
Time - column corresponding to datetime should have index 0 (leftmost). If the dataset datetime format is standard, you can leave the field Date format blank.
Volume - zipline use the column for calculations
All the other columns of the datasets are up to you.
